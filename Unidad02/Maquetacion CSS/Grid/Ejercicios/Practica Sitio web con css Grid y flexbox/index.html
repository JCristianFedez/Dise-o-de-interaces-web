La edad de oro del videojuego español

El término edad de oro del software español hace referencia a una época, aproximadamente entre 1983 y 1992, durante la cual España llegó a ser, detrás de Reino Unido, uno de los mayores productores europeos de software de entretenimiento para máquinas de 8 bits, sobre todo para Spectrum. El desarrollo y comercialización de software español, en general, había sido prácticamente inexistente hasta esas fechas, y la publicación de los primeros videojuegos made in spain son el detonante para que se comience a hablar en el mundo del software español.
